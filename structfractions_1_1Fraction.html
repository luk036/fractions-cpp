Detailed Description

template<typename T>
class fractions::Fraction< T >

Fraction.

Fraction class - C++ port of Python's fractions.Fraction.

Example: 

* Fraction<int> f(1, 2);
* f.numer() = 1;
* f.denom() = 2;
*
Template Parameters
T
*    +-------+
*    |  a    |    =  Fraction<T>
*    | ---   |
*    |  b    |
*    +-------+
*
*    a = numerator
*    b = denominator
*

A direct C++ implementation of Python's fractions.Fraction class that closely mirrors the Python implementation, including optimized arithmetic algorithms from Knuth's TAOCP.

Key features: 

* - Exact rational number representation
* - Automatic normalization to lowest terms
* - Optimized GCD-based arithmetic
* - Overflow-safe operations with floating-point fallback
* - Full comparison and arithmetic operator support
*
Template Parameters
TInteger type for numerator and denominator (default: int64_t)

Computes the cross product of this fraction and another fraction.

The cross product is defined as: cross(a/b, c/d) = a*d - b*c

This can be used to compute the determinant of a 2x2 matrix with the two fractions as the entries on the diagonal.

Example:

cross(Fraction(1,2), Fraction(3,4)) = -1 cross(Fraction(1,2), Fraction(1,2)) = 0 cross(Fraction(1,2), Fraction(-1,2)) = 0

Parameters
[in]rhsThe right-hand fraction to compute the cross product with.
Returns
The computed cross product.
*    a     c
*   --- x ---  =  a*d - b*c
*    b     d
*
*    +-------+     +-------+
*    |  a    |     |  c    |
*    | ---   |  x  | ---   |  =  a*d - b*c
*    |  b    |     |  d    |
*    +-------+     +-------+
*
*    cross(1/2, 3/4) = 1*4 - 2*3 = -2
